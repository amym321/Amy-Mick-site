Face Recognition App

Challenge - create an app utilizing an image recognition API, allow users to register and sign-in, and keep count of images.
Solution - a react app with a front-end, a backend server, and a PostgreSQL database. The frontend loads images and collects register/sign-in information. The backend routes sign-in information to the database and API responses. Server was built with Node for non-blocking runtime, and Express for http requests.
Result - a stylish frontend with full encryption. Based on a class project.

React, PostgreSQL, Express, Node.js
